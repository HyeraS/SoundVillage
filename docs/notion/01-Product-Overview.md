# 제품 개요

## 제품 목적

SoundMimic Village는 정답이 정해진 라벨을 고르는 대신, 참여자가 소리를 자기 언어의 의성어로 표현하도록 유도하는 데이터 수집 경험이다. 게임형 탐험, 반복 청취, 자신감 기록, 타인 표현 투표를 결합해 표현 데이터와 사회적 공감 신호를 함께 수집한다.

## 핵심 가치

- 자유 표현: 최대 80자의 의성어를 직접 입력한다.
- 맥락 제공: 소리를 Forest, Water, City, Music, Mystery Zone으로 묶는다.
- 참여 유도: 픽셀 아트 월드맵과 수집 경험으로 어노테이션 작업을 게임화한다.
- 품질 신호: 자신감, 재생 횟수, 청취 시간, 공감 투표를 함께 저장한다.
- 다기기 입력: 키보드 방향키/WASD와 화면 D-Pad를 지원한다.

## 대상 사용자

| 사용자 | 목표 |
|---|---|
| 연구 참여자 | 소리를 듣고 자연스러운 의성어를 기록 |
| 연구 운영자 | 참여자/세션별 데이터를 수집하고 품질을 관찰 |
| 개발자 | 사운드 카탈로그, UI, Supabase 계약을 유지보수 |

## 주요 기능

| 기능 | 현재 구현 |
|---|---|
| 참여자 식별 | 참여자 ID와 세션 ID를 직접 입력 |
| 월드 탐험 | 40×30 타일 SVG 월드맵, 5개 Zone 포털 |
| Zone 탐험 | 24×18 타일 SVG 맵, Zone별 배경과 오브젝트 |
| 소리 수집 | 아이템 충돌 시 자동 수집 및 패널 표시 |
| 소리 재생 | Howler.js, WAV/MP3/OGG 순차 후보 |
| Stage 1 | 표현, 자신감, 재생 횟수, 청취 시간 저장 |
| Stage 2 | 같은 소리의 다른 표현 최대 6개 조회 및 복수 투표 |
| 진행도 | 전체 어노테이션 수와 Zone별 진행바 |
| 완료 피드백 | 수집 토스트와 Zone 전체 수집 완료 모달 |

## Zone 정의

| Zone | 한글 라벨 | 대표 범주 | 강조색 |
|---|---|---|---|
| Forest | 숲속 마을 | Animals, Wind | `#5B9E3A` |
| Water | 물가 마을 | Flow, Impact | `#4A8FD4` |
| City | 도시 마을 | Vehicle, Crowd | `#C4B99A` |
| Music | 음악 마을 | Melodic, Percussive | `#9B6DD4` |
| Mystery | 미스터리 마을 | Synthetic, Abstract | `#D4883A` |

## 성공 지표 후보

현재 코드가 직접 수집하거나 계산 가능한 지표다.

- 전체 비스킵 어노테이션 수
- Zone별 비스킵 어노테이션 수
- 소리별 표현 수와 투표 수
- 참여자/세션별 제출 수
- 표현별 자신감
- Stage별 재생 횟수와 청취 시간
- 스킵 비율과 스킵 사유

## 현재 범위 밖

- 로그인, 인증, 관리자 화면
- 참여자별 진행도 영속화
- 실험 조건 배정 및 세션 검증
- 사운드 파일 업로드/관리 UI
- 데이터 분석 대시보드
- 자동 테스트
- 접근성 완전 지원
