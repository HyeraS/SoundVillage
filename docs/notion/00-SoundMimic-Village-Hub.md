# SoundMimic Village

> 소리를 탐험하고, 의성어로 표현하고, 다른 참여자의 표현에 공감 투표하는 웹 기반 사운드 어노테이션 게임.

## 문서 정보

| 항목 | 값 |
|---|---|
| 문서 기준일 | 2026-06-05 |
| 구현 버전 표기 | `v0.4-web` |
| 실제 앱 루트 | `soundmimic-village/` |
| 기술 스택 | Next.js 16, React 19, Supabase, Howler.js |
| 현재 상태 | 프로덕션 빌드 성공, ESLint 오류 10건 |

## 한눈에 보기

참여자는 ID와 세션 ID를 입력하고 픽셀 아트 월드맵에 입장한다. 다섯 Zone 중 하나에 들어가 소리 아이템과 충돌하면 어노테이션 패널이 열린다. 참여자는 소리를 듣고 자유로운 의성어와 자신감을 제출한 뒤, 같은 소리에 대해 다른 참여자가 남긴 표현에 복수 투표할 수 있다. 결과는 Supabase의 `annotations`, `votes` 테이블에 저장된다.

## Notion 권장 구성

이 파일을 최상위 페이지로 만들고 나머지 문서를 하위 페이지로 가져온다.

| Notion 데이터베이스 | 권장 속성 |
|---|---|
| 프로젝트 문서 | 이름, 문서 유형, 상태, 담당자, 최종 검토일, 관련 코드 |
| 의사결정 로그 | 결정, 날짜, 배경, 대안, 결과 |
| 이슈 및 로드맵 | 제목, 심각도, 상태, 담당자, 관련 컴포넌트 |
| 사운드 카탈로그 | `sound_id`, Zone, 하위 분류, 파일 경로, 원본 데이터셋, 검증 상태 |

## 핵심 용어

| 용어 | 의미 |
|---|---|
| Zone | Forest, Water, City, Music, Mystery 중 하나인 소리 테마 마을 |
| Sound item | ZoneMap에서 충돌해 수집하는 소리 오브젝트 |
| Stage 1 | 소리를 듣고 의성어와 자신감을 제출하는 단계 |
| Stage 2 | 다른 참여자의 표현에 공감 투표하는 단계 |
| 수집 완료 | 현재 브라우저 세션에서 해당 소리를 제출하거나 건너뛴 상태 |
| 전역 진행도 | Supabase에 저장된 Zone별 비스킵 어노테이션 수를 10개 기준으로 환산한 값 |

## 중요한 구분

- `collectedIds`는 현재 React 메모리에만 존재하며 새로고침하면 초기화된다.
- 월드맵 진행도와 총 기록 수는 개인 진행도가 아니라 모든 참여자의 Supabase 집계다.
- 저장소에는 메타데이터가 가리키는 실제 오디오 파일이 현재 없다.
- 저장소 상위 디렉터리에도 과거 버전으로 보이는 중복 파일이 있으나 실제 앱은 `soundmimic-village/` 내부 파일을 사용한다.
